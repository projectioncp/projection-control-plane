# Roadmap

## Overview

Projection is an experimental enterprise agentic control-plane prototype focused on:
- bounded operational context
- governed orchestration
- deterministic execution
- auditability
- safer enterprise AI interaction patterns

The roadmap below reflects the current exploration direction of the project.

---

# Phase 1 — Core Manufacturing MVP

Current focus.

## Objectives
- establish Projection Frame concepts
- implement orchestration flow
- support deterministic capability execution
- enable audit lifecycle capture
- demonstrate bounded operational reasoning

## Initial Areas
- manufacturing workflows
- supply-chain workflows
- BOM/context retrieval
- workflow orchestration
- operational state tracking
- governed execution patterns

## Initial Components
- orchestrator
- capability runtime
- retrieval layer
- audit logging
- workflow hooks
- operational state model
